USA Weightlifting's American Open 3

Hope Stremcha MAP671

The 2018 Nike American Open 3 sponsored by Rogue was a weightlifting meet of historic proportions. With over 1600 lifters, USA Weightlifting has applied for a Guiness World record for the largest Olympic Weightlifting Competition ever held. The athletes competing represented over 300 registered clubs, 298 of which had valid addresses listed on either the USA Weightlifting or each gym's website. The 298 clubs and their density are shown on the map above.

Weightlifting clubs present at the competition were extracted from the start list on the Competition Website

Tools used in this project

  • Data was created using OpenStreetMap.
  • QGIS 2.18.21 QuickOSM to create weightlifting club layer.
  • QGIS 2.18.21 heatmap plugin to create heatmap.
  • Map CRS is EPSG:102008 North America Albers Equal Area Conic
